# React Aggregate ERC20 Swap

![Modern Neon Sci-Fi Fantasy Book Cover  copy](https://github.com/FanbaseLabs/React-Aggregate-ERC20-Swap/assets/145924938/277ca724-0bf4-4815-a4f9-12a26d467a69)

This project is a Create React App template that embeds the `fanbase-aggregate-erc20-swap-widget`.

It renders a single aggregate ERC20 swap widget in `src/App.tsx` with:
- a fixed `appId`
- `chainId={80001}`

Documentation:
- Fanbase aggregate swap widget docs: https://fanbase-io.gitbook.io/docs/reference/swap-widget-sdk/aggregate-swap-widget
- Fanbase plans: https://www.plan.fanbase.io

![PM-3](https://github.com/FanbaseLabs/React-Aggregate-ERC20-Swap/assets/145924938/58fb1d00-438f-494d-9c18-53dbc6549c1d)

## Scripts

In the project directory:

- `npm start` - run the app in development mode at `http://localhost:3000`
- `npm test` - run tests in watch mode
- `npm run build` - create a production build in `build/`
- `npm run eject` - eject CRA configuration (irreversible)

## Tech Stack

- React 18 + TypeScript
- `react-scripts` (Create React App)
- `fanbase-aggregate-erc20-swap-widget`
